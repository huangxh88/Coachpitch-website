CoachPitch Static Site Package

Files included:
- index.html
- faq.html
- privacy.html
- styles.css
- assets/ (add your images here)

Image filenames expected by the homepage:
- assets/screenshot-phone-1.svg
- assets/screenshot-phone-2.svg
- assets/screenshot-phone-3.svg
- assets/screenshot-phone-4.svg
- assets/screenshot-watch-1.svg
- assets/screenshot-watch-2.svg
- assets/screenshot-watch-3.svg
- assets/screenshot-watch-4.svg

How to use:
1. Copy these files into your GitHub repository root.
2. Add your real screenshots/mockups into the assets folder using the filenames above.
3. Update copy and contact details in the HTML files as needed for your launch.
4. Replace hello@coachpitch.app with your real contact email if needed.
5. Deploy the repo to Render as a Static Site.

Note:
The chat images were not available as binary files in the workspace, so the site is fully wired for your screenshots but you will need to drop the files into /assets yourself.
